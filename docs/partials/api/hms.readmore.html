readmore
service in module hms

Description

Add this attribute to make an element (use a div) containing 'read more' information. Any HTML placed inside as children will be hidden until "Read More.." is clicked

  • rid: Must be a unique ID
  • type: 'readmore' or 'task'
  • title: Title of the box 
    <div readmore rid="rm1" title="This is my title" type="readmore"></div>

Usage

as attribute
<ANY readmore>
   ...
</ANY>
