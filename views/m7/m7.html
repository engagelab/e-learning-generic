An attribute is extra functionality added to an existing HTML tag e.g. 

<div hmsquickquestion h-question="Did you know?" h-answer="No I didn't!"></div>

In that example 'hmsquickquestion', 'h-question' and 'h-answer' are attributes added to a div tag


Make sure to include values for all the attributes in your widget code. At this point not doing so may cause unexpected behaviour





An element is a new HTML tag e.g. 

<hmsquiz></hmsquiz>

In that example 'hmsquiz' is an element created especially for the HMS project, and includes lots of functionality

readmore
Add this attribute to make an element (use a div) containing 'read more' information. The 'rid' must be unique for the page. 'type' can be 'readmore' or empty to show the "read more" icon. Use "task" to show a notepad icon. Any HTML placed inside as children will be hidden until "Read More.." is clicked (attribute)
<div hmsreadmore rid="rm1" title="Click me to read more.." type="readmore" ></div>
Here is the content inside!
readmore (task)
<div readmore rid="rm2" title="This readmore has a task icon" type="task" ></div>
Here is the content inside!
